Backends

DifferentiationInterface.ChainRulesForwardBackend — Type
ChainRulesForwardBackend{RC}

Performs autodiff with forward-mode AD packages based on ChainRulesCore.jl, like Diffractor.jl.

This must be constructed with an appropriate RuleConfig instance.

using Diffractor, DifferentiationInterface
backend = ChainRulesForwardBackend(Diffractor.DiffractorRuleConfig())
source
DifferentiationInterface.ChainRulesReverseBackend — Type
ChainRulesReverseBackend{RC}

Performs autodiff with reverse-mode AD packages based on ChainRulesCore.jl, like Zygote.jl.

This must be constructed with an appropriate RuleConfig instance:

using Zygote, DifferentiationInterface
backend = ChainRulesReverseBackend(Zygote.ZygoteRuleConfig())

Primitives

Pushforward

DifferentiationInterface.value_and_pushforward! — Method
value_and_pushforward!(dy, backend, f, x, dx[, stuff]) -> (y, dy)

Compute the Jacobian-vector product of the Jacobian of f at x with dx inside dy. Returns the primal output f(x) and the JVP dy.

Interface requirement

This is the only required implementation for a forward mode backend.

Arguments

  • y: primal output
  • dy: cotangent, might be modified
  • backend: forward-mode autodiff backend
  • f: function x -> y to differentiate
  • x: argument
  • dx: tangent
  • stuff: optional backend-specific storage (cache, config), might be modified
source

Pullback

DifferentiationInterface.value_and_pullback! — Method
value_and_pullback!(dx, backend, f, x, dy[, stuff]) -> (y, dx)

Compute the vector-Jacobian product of dy with the Jacobian of f at x inside dx. Returns the primal output f(x) and the VJP dx.

Interface requirement

This is the only required implementation for a reverse mode backend.

Arguments

  • y: primal output
  • dx: tangent, might be modified
  • backend: reverse-mode autodiff backend
  • f: function x -> y to differentiate
  • x: argument
  • dy: cotangent
  • stuff: optional backend-specific storage (cache, config), might be modified

Array to array

DifferentiationInterface.value_and_jacobian! — Method
value_and_jacobian!(jac, backend, f, x[, stuff]) -> (y, jac)

Compute the Jacobian inside jac. Returns the primal output f(x) and the Jacobian.

Notes

For a function f: ℝⁿ → ℝᵐ, jac is expected to be a m × n matrix. If the input or output is a higher-order array, it is flattened with vec.
